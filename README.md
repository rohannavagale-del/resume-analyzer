Project Overview :

The Resume Analyzer is a web-based application that helps analyze resumes and compare them with job descriptions. It uses basic Natural Language Processing (NLP) techniques to extract skills .
and keywords from resumes and provides a match score along with suggestions for improvement. The project is designed to assist students and job seekers in understanding how well their resume aligns with a specific role

Objectives:

Extract key skills and keywords from resumes

Compare resumes with job descriptions

Generate a similarity or match score

Provide basic feedback to improve resumes

Tech Stack :

Programming Language: Python

Backend Framework: Flask / FastAPI

AI / NLP: TF-IDF, Cosine Similarity, spaCy / NLTK

Libraries: scikit-learn, Pandas, NumPy

Frontend: HTML, CSS, JavaScript

Version Control: Git & GitHub

 AI Concept Used :

The project uses NLP techniques such as text preprocessing, keyword extraction, and TF-IDF with cosine similarity to measure how closely a resume matches a given job description.

 How It Works

User uploads a resume

Resume text is cleaned and processed

Skills and keywords are extracted

Resume is compared with the job description

A match score and suggestions are displayed
